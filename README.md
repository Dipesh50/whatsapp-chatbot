# whatsapp-chatbot
# WhatsApp Chatbot Backend (Spring Boot)

## Features
- REST API endpoint: /webhook
- Accepts JSON input
- Responds with predefined replies
- Logs incoming messages

## API Usage

### Endpoint:
POST /webhook

### Request:
{
  "message": "hi"
}

### Response:
Hello

## Tech Stack
- Java
- Spring Boot
- REST API

## How to Run
1. Clone repo
2. Run application
3. Test using Postman
